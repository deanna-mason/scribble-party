# Scribble Party

Real-time multiplayer collaborative drawing game for 2–8 players. Built with Express, `ws`, and vanilla JS Canvas. Deployed to Render.

## Running locally

```bash
npm install
npm start
```

Open <http://localhost:3000>.

## Running tests

```bash
npm test
```
